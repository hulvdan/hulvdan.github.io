Clip Studio. Быстрый экспорт «по кнопке». AutoHotkey макрос

Проблема

В Clip Studio нет Keyboard Shortcut-а для мгновенного экспорта файла в png

Из-за этого мой flow работы выглядел так:

  1. Внес исправление в картинку
  2. Навел мышку на «File»
  3. Ткнул
  4. Навел мышку на «Export Multiple Pages» (я спиратил Clip Studio Ex и использую функционал для мангак, чтобы на разных страницах «манги» рисовать разные spritesheet-ы. Каждая страница разбивается на отдельный файл ➜ VCS меньше нагружается)
  5. Навел мышку на «.png»
  6. Ткнул
  7. Ввел название
  8. Нажал Enter 3 раза
